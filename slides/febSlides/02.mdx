# More Docker Benefits

- Scalable: Extremely lightweight can be easily launched and shutdown
- Portable: Take snapshots of your environment and upload it
- Density: Efficient use of resources
- Deployment: Can easily run anywhere for apps built in any language

- Overall : Dramatically reduces development lifecycle

# Docker Components

- Docker Daemon : Docker Engine that runs on the host machine

- Docker Client: CLI used to interact with the Daemon
