# What is Docker?

- Docker is an open source software platform to create, deploy, and manage virtualized application containers on a common operating system
- Separates an application's infrastructure using containers
- Similar to how Virtual Machines sepaate operating systems from hardware

# What is a container?

- Allows a developer to package an application and all of its parts in a "box / container". (Stack, Dependencies)
- Isolated environment where the application has everything it needs to run
- The operating system is abstracted

# What problem does this solve?

- Dependency hell
- Everyone has different environments (Developers, QA, SysAdmin)
- Helps with Continous Integration / Delivery, portability...

# Why are containers popular?

- Docker
- Makes using containers easier
